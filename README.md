## Features

- [mailwind](https://github.com/soheilpro/mailwind) Use Tailwind CSS to design HTML emails.

## Install

```sh
yarn install
```

## Usage

```sh
yarn dev
yarn dev:inline
yarn mailwind
yarn mailwind:inline
```
